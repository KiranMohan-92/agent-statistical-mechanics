  <action>
    Derive critical exponents from RG eigenvalues and establish finite-size scaling.

    **Part 1: Exponent Definitions and RG Derivation**

    Primary exponents from RG eigenvalues at critical FP:
    - ν = 1/y_t (correlation length)
    - η = anomalous dimension from field rescaling
    - y_h = (d + 2 - η)/2 (magnetic eigenvalue)

    Derived exponents:
    - β = (d - 2 + η)y_h / 2y_t (order parameter)
    - γ = (2 - η)y_h / y_t (susceptibility)
    - α = 2 - dν (specific heat)
    - δ = (d + 2 - η) / (d - 2 + η) (critical isotherm)

    **Part 2: Epsilon Expansion Results**

    For d = 4 - ε:
    - η = 0 + O(ε²)
    - ν = 1/2 + ε/12 + O(ε²)
    - β = 1/2 - ε/6 + O(ε²)
    - γ = 1 + ε/6 + O(ε²)

    Numerical values:
    - d=3 (ε=1): ν ≈ 7/12 ≈ 0.58, β ≈ 1/3, γ ≈ 7/6 ≈ 1.16
    - d=2: Use exact Baxter results (not epsilon expansion)

    **Part 3: Exact 2D Results (Baxter)**

    For d=2 Potts model (q ≤ 4):
    - g = 4/√q
    - β = (g - 2)/(12g)
    - ν = √3/(2√g)
    - γ = (7g - 12)/(6g)
    - η = 1/4 (Ising), varies with q

    Table for reference:
    | q | β | ν | γ | η |
    |---|-----|-----|-----|-----|
    | 2 | 1/8 | 1 | 7/4 | 1/4 |
    | 3 | 1/9 | 5/6 | 13/9 | 4/15 |
    | 4 | 1/12 | 2/3 | 7/6 | 1/4 |

    **Part 4: Scaling Relations**

    Rushbrooke: α + 2β + γ = 2
    Widom: γ = β(δ - 1)
    Fisher: γ = ν(2 - η)
    Hyperscaling: 2 - α = dν (valid for d < 4)

    **Part 5: Finite-Size Scaling Forms**

    Define reduced temperature: t = (T - T_c)/T_c

    Scaling forms:
    - m(N, t) = N^{-β/dν} f_m(t N^{1/dν})
    - χ(N, t) = N^{γ/dν} f_χ(t N^{1/dν})
    - ξ(N, t) = N^{1/d} f_ξ(t N^{1/dν})

    At T_c (t = 0):
    - m(N, 0) ∼ N^{-β/dν}
    - χ(N, 0) ∼ N^{γ/dν}
    - ξ(N, 0) ∼ N^{1/d}

    **Part 6: N* Scaling Near T_c**

    From Phase 2, mean-field N* = (T/2ε) ln[e^{J/2T} + (q-1)e^{-J/2T}]

    Critical corrections:
    - Near T_c, correlation length ξ ∼ |t|^{-ν}
    - When ξ ≳ N^{1/d}, finite-size effects become important
    - Modified N*: N*_finite = N*_MF × [1 + O(N^{-1/dν})]

    Practical implications:
    - For N < 50: significant rounding of phase transition
    - For N < 20: T_c can shift by 20-30%
    - Agent systems always in finite-size regime
