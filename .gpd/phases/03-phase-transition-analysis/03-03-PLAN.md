  <action>
    Analyze the complete fixed point structure and establish universality.

    **Part 1: Fixed Point Taxonomy**

    Identify all fixed points of the RG flow:

    1. **High-Temperature (HT) Fixed Point:**
       - Coordinates: r* → +∞, u* → 0
       - Physical meaning: Complete disorder, infinite temperature
       - Stability: Stable in disordered phase (T > T_c)
       - Eigenvalues: One irrelevant (thermal)

    2. **Low-Temperature (LT) Fixed Point:**
       - Coordinates: r* → -∞, u* → 0
       - Physical meaning: Complete order, zero temperature
       - Stability: Stable in ordered phase (T < T_c)
       - Eigenvalues: One irrelevant (thermal)

    3. **Gaussian Fixed Point:**
       - Coordinates: r* = 0, u* = 0
       - Physical meaning: Free field theory, mean-field critical
       - Stability: Stable for d > 4, unstable for d < 4
       - Eigenvalues: y_r = 2, y_u = 4-d

    4. **Wilson-Fisher Fixed Point:**
       - Coordinates: r* = O(ε) < 0, u* = O(ε) > 0
       - Physical meaning: Non-trivial critical behavior
       - Stability: Stable for d < 4
       - Eigenvalues: y_t = 2 - ε/3, y_h = (d+2)/2 + ...

    **Part 2: Basin of Attraction**

    - Identify separatrix between HT and LT basins
    - Critical surface: r = r_c(u) flows to critical FP
    - Relevance classification:
      * Relevant: perturbation grows under RG (y > 0)
      * Irrelevant: perturbation shrinks (y < 0)
      * Marginal: y = 0, requires higher-order analysis

    **Part 3: Universality Argument**

    1. **Microscopic parameters:** J (coupling), lattice structure, etc.
    2. **RG flow:** All microscopic details flow to FP
    3. **Critical exponents:** Determined by FP eigenvalues only
    4. **Universality classes:** Same d + symmetry → same exponents

    **Part 4: q-Dependence and q=4 Threshold**

    For d=2 Potts model, exact results from Baxter:

    | q | Transition | β | ν | γ | η |
    |---|-----------|---|---|---|---|
    | 2 | 2nd order | 1/8 | 1 | 7/4 | 1/4 |
    | 3 | 2nd order | 1/9 | 5/6 | 13/9 | 4/15 |
    | 4 | Marginal | 1/12 | 2/3 | 7/6 | 1/4 |
    | > 4 | 1st order | - | - | - | - |

    Conformal field theory formulas (Nienhuis, 1982):
    g = 4/√q for q ≤ 4
    β = (g - 2)/(12g)
    ν = √3/(2√g)

    **Part 5: Agent System Implications**

    - q ≤ 4: Gradual consensus (continuous m(T) near T_c)
    - q > 4: Abrupt consensus (discontinuous jump in m)
    - For Yang's D_4, D_8, D_16:
      * D_4 (q=4): Marginal case, smooth but with critical slowing
      * D_8, D_16 (q > 4): First-order, abrupt consensus at T_c
