  <name>Task 1: Derive RG flow equations for Potts model</name>
  <files>derivations/rg-flow-equations.md</files>
  <action>
    Derive the momentum-shell renormalization group flow equations.

    **Starting point: Landau-Ginzburg-Wilson free energy functional**
    For the Ising case (q=2), use scalar φ⁴ theory:
    F[φ] = ∫ d^d x [½(∇φ)² + ½rφ² + uφ⁴]

    For general q, note that the Potts model maps to a vector field theory
    (document the mapping but focus on Ising for explicit derivation).

    **Step 1: Momentum-shell integration**
    - Split field into slow and fast modes: φ = φₛ + φ_f
    - Slow modes: |k| < Λ/b
    - Fast modes: Λ/b < |k| < Λ
    - Integrate out fast modes: Z = ∫ Dφ_f exp(-F[φₛ + φ_f])

    **Step 2: One-loop correction to free energy**
    Compute the Gaussian integral over φ_f to O(u):
    ΔF = -½ Tr ln(δ²F/δφ²)

    **Step 3: Renormalization**
    - Rescale coordinates: x' = x/b
    - Rescale field: φ'(x') = b^{(d-2+η)/2} φₛ(x)
    - Read off new parameters r', u'

    **Step 4: Flow equations**
    For infinitesimal transformation b = e^{dl} ≈ 1 + dl:
    dr/dl = 2r + 12u K_d ∫_0^Λ k^{d-1} dk / (k² + r) + ...
    du/dl = (4-d)u - 36u² K_d ∫_0^Λ k^{d-1} dk / (k² + r)² + ...

    where K_d = S_d/(2π)^d is the phase space factor and S_d = 2π^{d/2}/Γ(d/2).

    **Step 5: Evaluate integrals**
    At one-loop, evaluate the momentum integrals:
    I₁ = K_d Λ^d / (1 + r/Λ²)
    I₂ = K_d Λ^d / (1 + r/Λ²)²

    **Step 6: Fixed point analysis**
    Gaussian fixed point: r* = 0, u* = 0
    Linearize to find eigenvalues: y_r = 2, y_u = 4-d

    Wilson-Fisher fixed point: Set dr/dl = du/dl = 0
    Solve for r* and u* to O(ε):
    r* = -K_d Λ² ε/6 + O(ε²)
    u* = (4-d)/(36 K_d Λ^{d-4}) + O(ε²)

    **Step 7: Critical exponents**
    From eigenvalues at Wilson-Fisher:
    y_t = 2 - ε/3 + O(ε²) → ν = 1/y_t = 1/2 + ε/12 + ...
    y_h = (d+2-η)/2 → η = O(ε²)
    β = (d-2+η)y_h/2y_t → β = 1/2 - ε/6 + ...
    γ = (2η-d)y_h/y_t → γ = 1 + ε/6 + ...

    Include ASSERT_CONVENTION at the top of the derivation file.
  </action>
