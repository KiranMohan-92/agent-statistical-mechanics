  <files>.gpd/phases/02-partition-function-derivation/02-03-free-energy.md</files>
  <action>
Calculate free energy from the mean-field partition function:

1. **Starting point:**
   Z_MF ≈ [e^{βJ} + (q-1)]^{N-1} × [e^{βJ(N-1)} + (q-1)q]

2. **Free energy:**
   F = -T ln Z
   ln Z = (N-1) ln[e^{βJ} + (q-1)] + ln[e^{βJ(N-1)} + (q-1)q]

3. **Simplified form (thermodynamic limit N → ∞):**
   F ≈ -NT ln[e^{βJ} + (q-1)]
     - Per-agent: f = F/N = -T ln[e^{βJ} + (q-1)]

4. **Interpretation:**
   - First term: -NT ln e^{βJ} = -NJ (alignment energy)
   - Second term: -NT ln(q-1) (diversity entropy)
   - Trade-off: alignment favors fewer agents, diversity favors more agent types

Write free energy expression with physical interpretation of each term.
  </action>
  <verify>
- [F] = [energy] (consistent with thermodynamic definition)
- F → -∞ as T → 0 (perfect alignment)
- F diverges as q → ∞ (too many states)
- Mean-field free energy structure matches Goldenfeld (1992)
  </verify>
