## Task 5: Calculate entropy and internal energy

**Started:** 2026-03-18T21:49:00Z
**Completed:** 2026-03-18T21:52:00Z

### Actions
- Computed internal energy U = -d(ln Z)/d(beta)
- Computed entropy S = beta U + ln Z
- Computed heat capacity C = dU/dT
- Derived free energy F = -T ln Z

### Key Results
- U = -(NJ/2) * [exp(x) - (q-1)exp(-x)] / [exp(x) + (q-1)exp(-x)] where x = beta J/2
- S expressed in terms of U and ln Z
- C = N (beta J)^2 q exp(beta J) / [exp(beta J/2) + (q-1)exp(-beta J/2)]^4
- F = -(N/beta) * ln[exp(beta J/2) + (q-1)exp(-beta J/2)]

### Verification
- [x] U = -d(ln Z)/d(beta) correctly applied
- [x] S = beta U + ln Z matches thermodynamic identity
