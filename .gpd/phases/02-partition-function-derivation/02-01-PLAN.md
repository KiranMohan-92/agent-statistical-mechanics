  <name>Task 2: Define interaction topology and coupling J</name>
  <files>.gpd/phases/02-partition-function-derivation/02-01-derivation.md</files>
  <action>
Define the interaction structure between agents:

1. **Interaction pairs ⟨ij⟩:**
   - For mean-field: all agent pairs interact (fully-connected)
   - Sum: Σ_{⟨ij⟩} → (N/2) Σ_{i≠j} for undirected pairs

2. **Coupling strength J > 0:**
   - J = probability of agent agreement × interaction bandwidth
   - J > 0: ferromagnetic (alignment favored)
   - J captures communication quality, shared context, coordination protocols

3. **Kronecker delta δ(s_i, s_j):**
   - δ(s_i, s_j) = 1 if agents i, j have same state
   - δ(s_i, s_j) = 0 if agents have different states
   - Measures alignment/compatibility between agents

4. **Hamiltonian sign:** H = -J Σ δ(s_i, s_j)
   - Negative sign: lower energy = higher alignment
   - Lower H corresponds to better task performance

  <name>Task 3: Define effective temperature T_eff</name>
  <files>.gpd/phases/02-partition-function-derivation/02-01-derivation.md</files>
  <action>
Define the temperature parameter and its interpretation:

1. **Boltzmann factor:** β = 1/T_eff with k_B = 1 (natural units)

2. **Operational definition:**
   T_eff = (disagreement rate) / (coupling strength)
   - High T_eff: agents frequently disagree (disordered phase)
   - Low T_eff: agents strongly align (ordered phase)

3. **Alternative (entropy-based):**
   T_eff = S where S = -Σ p_k ln p_k
   - p_k = probability of agent being in state k
   - Measures diversity of agent responses

4. **Physical meaning:**
   - T_eff captures noise/stochasticity in LLM responses
   - Relates to task ambiguity, prompt variability, model stochasticity
   - Balances energy minimization (alignment) vs. entropy maximization (diversity)

Establish the relationship between Hamiltonian energy and task performance:

1. **Hypothesis:**
   Performance P ∝ exp(-βH) / Z
   - Low H (high alignment) → High performance
   - High H (disagreement) → Low performance

2. **Physical interpretation:**
   - Energy H represents cost/misalignment in agent system
   - Alignment (low H) enables efficient coordination
   - Disagreement (high H) causes conflicts and inefficiency

3. **Free energy interpretation:**
   F = U - TS
   - U = -J Σ δ (alignment energy)
   - S = ln Ω (diversity entropy)
   - Optimal N* minimizes F (balance of alignment and diversity)

4. **Connection to Yang et al.:**
   - Diverse systems (high q) have higher entropy S
   - Entropy can compensate for lower alignment energy
   - Explains "2 diverse ≈ 16 homogeneous" as free energy equivalence
