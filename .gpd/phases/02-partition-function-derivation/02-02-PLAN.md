Establish the foundation for partition function calculation:

1. **Hamiltonian:**
   H = -J Σ_{⟨ij⟩} δ(s_i, s_j)
   where s_i ∈ {1, ..., q}, J > 0

2. **Mean-field topology:**
   - Fully-connected: all N agents interact with all others
   - Number of pairs: N(N-1)/2
   - Energy for configuration {n_1, ..., n_q}: E = -J/2 Σ_{k=1}^q n_k (n_k - 1)
     where n_k = number of agents in state k, Σ n_k = N

3. **Partition function definition:**
   Z = Σ_{states} e^{-βH} = Σ_{n_1+...+n_q=N} [N!/(n_1! ... n_q!)] × exp(βJ/2 Σ_k n_k (n_k - 1))

4. **Degeneracy factor:**
   - Multinomial coefficient N!/(n_1! ... n_q!) counts arrangements
   - For distinguishable agents with states

Write the setup with clear explanation of the mean-field approximation.

<task type="auto">
  <name>Task 2: Derive mean-field partition function</name>
  <files>.gpd/phases/02-partition-function-derivation/02-02-partition-function.md</files>
  <action>
Derive the closed-form partition function under mean-field:

1. **Approach 1: Order parameter method**
   - Define order parameter m = (q N_max - N)/[(q-1)N]
   - Mean-field free energy: F = -NT ln q - (NJ/2) m^2 + NT S(m)
   - Stationary phase approximation for Z ≈ e^{-βF}

2. **Approach 2: Direct summation (preferred for finite N)**
   - Use recursion or integral representation
   - For fully-connected: Z = Σ_{k=1}^q e^{βJ N k^2/N} × (multinomial count)
   - Simplify using dominant state approximation

3. **Final mean-field result:**
   Z_MF ≈ [e^{βJ} + (q-1)]^{N-1} × [e^{βJ(N-1)} + (q-1)q]

   Or in the thermodynamic limit:
   Z_MF/N ≈ [e^{βJ} + (q-1)]^N for large N

4. **Free energy:**
   F = -T ln Z
   Per-agent: f = -T/N ln Z

Derive both the exact finite-N form and the thermodynamic limit approximation.
  </action>

- Z → [e^{βJ} + (q-1)]^N as N → ∞ (correct thermodynamic limit)
- F = -T ln Z has dimensions of energy
- For q=2 (Ising), result reduces to known mean-field Ising partition function
  </verify>
